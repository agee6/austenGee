Lit Lives

A website for exploring literature and interacting with it. Keep track of books you have read and notes you have made. I built this from the ground up, from conceptualization to deployment. It uses a postgres database which uses Active Record and Ruby and Rails for the backend. The front end is a single page app built with React and jQuery. Heroku deployment. Integrates with Google books api.
